## Installation

```bash
git clone https://github.com/lukechilds/zsh-nvm ~/.oh-my-zsh/custom/plugins/zsh-nvm
```

`plugins+=(zsh-nvm)`

## You can list available versions using ls-remote:

```bash
nvm ls-remote
1
```

```bash
nvm install --lts
```
